Claude-meter

Claude Code Limity, vždy na očích.

Rozšíření pro VS Code, které zobrazuje limity využití Claude Code (5hodinová relace a týdenní) jako trvalou položku ve stavovém řádku — vždy viditelné, bez nutnosti zadávat příkazy.

Navrženo a řízeno mnou, kód napsán s pomocí AI. Všechna rozhodnutí o funkcích, přístupy a produktová vize jsou moje vlastní.

Claude Code JS JSON VS Code GitHub VS Marketplace
Více o projektu a vývoji

Záměr projektu

  • Vznikl z reálné osobní frustrace — žádný nativní způsob, jak vidět limity Claude Code přímo ve VS Code
  • Záměrně minimalistické — jedno JS rozšíření, žádný build step

Technický přehled

  • Typ: Rozšíření pro VS Code — publikováno na VS Code Marketplace
  • Jazyk: Vanilla JavaScript, VS Code Extension API
  • Auth: Čte OAuth přihlašovací údaje z ~/.claude/.credentials.json vygenerované Claude Code
  • Dotazuje api.anthropic.com/api/oauth/usage každých 5 minut (endpoint objevený komunitou)
  • Barevně označený stavový řádek — šedá / žlutá / červená podle procenta využití
  • Tooltip při najetí zobrazuje kvóty relace i týdne, odpočty do resetu a čas posledního načtení
  • Žádná data se neshromažďují — pouze jeden dotaz na servery Anthropic pomocí vlastního tokenu

Aktuální stav

  • Dostupné na VS Code Marketplace a v aktivním osobním použití
  • Funguje automaticky při spuštění VS Code bez nutnosti konfigurace
  • Závisí na nedokumentovaném API endpointu — může přestat fungovat, pokud jej Anthropic změní

GearUp: Your digital armory

Aplikace pro správu vybavení vytvořená s pomocí AI.

Mobilní Progressive Web App (PWA) pro správu airsoft vybavení, generování loadoutů na mise a sledování gearu přes více zařízení. Navržena v souladu s vizuální identitou mého airsoft týmu RAT PMC — tmavá monochromatická estetika, instalovatelná na telefon i počítač, funguje plně offline.

Claude Cowork JS HTML CSS GitHub Cloudflare Supabase
Více o projektu a vývoji

Navrženo a řízeno mnou, kód napsán s pomocí AI. Všechna rozhodnutí o funkcích, UX design a produktová vize jsou moje vlastní.

Záměr projektu

  • Vznikla z reálné osobní potřeby — správa gearu a vybavení pro airsoft
  • Záměrně bez frameworků — čisté soubory, snadná údržba

Technický přehled

  • Typ: Progressive Web App (PWA) — instalovatelná, funguje offline
  • Frontend: Vanilla JS, HTML, CSS — bez frameworků
  • Úložiště: IndexedDB (lokálně) + Supabase PostgreSQL (cloud sync)
  • Auth: Cloudflare Access (ověření před načtením jakéhokoli souboru)
  • Generátor loadoutů se skórovacím algoritmem (váha sezóny, účelu a stavu vybavení)
  • 6 stavů vlastnictví — vlastní, zapůjčené, půjčené (někomu), na prodej, wishlist, prodáno
  • Import/export Excel a JSON s automatickým rozpoznáváním sloupců a slučováním duplicit
  • Stránka statistik s Chart.js grafy — typ, stav, sezóna, účel
  • Přepínač měny CZK / EUR s převodem cen napříč všemi předměty

Aktuální stav

  • Plně funkční, v denním osobním použití
  • Sync mezi zařízeními přes Supabase s JWT authentication a automatickým obnovením tokenů
  • Nasazena na Cloudflare Pages, soukromý přístup přes Cloudflare Access

Bohunice: The Last Shift

Můj vlastní dlouhodobý solo projekt — aktivně rozpracovaný

First-person, atmosférická hra která se odehrává po zfiktované jaderné katastrofě na území Československa v jaderné elektrárně Jaslovské Bohunice v roce 1977, zaměřená na napětí, navigaci a příběh prostřednictvím prostředí kolem hráče.

Vytvořeno samostatně během několika měsíců. Veškeré modely, textury a level design jsou mojí vlastní prací.

Směr projektu

  • Původně plánováno s mnohem širším záběrem
  • Hodně funkcí bylo cíleně zjednodušeno nebo odstraněno, aby byl projekt soustředěný a zvládatelný
  • Důraz na atmosféru.

  • Pohled: First-person
  • Vizualní styl: Low-poly / PSX inspirace

Současný stav
