Claude-meter: Limits at a glance

Claude Code usage limits, always visible.

A VS Code extension that displays your Claude Code usage limits (5-hour session and weekly) as a permanent status bar item — always visible, no commands needed.

Designed and directed by me, code written with AI assistance. All feature decisions, limits access, and product direction are my own.

Claude Code JS JSON VS Code GitHub VS Marketplace
Read more about the project and development

Project Direction

  • Built to solve a real personal frustration — no native way to see Claude Code usage limits inside VS Code
  • Intentionally lightweight — a single JS extension

Technical Overview

  • Type: VS Code Extension — published on the VS Code Marketplace
  • Language: Vanilla JavaScript, VS Code Extension API
  • Auth: Reads OAuth credentials from ~/.claude/.credentials.json generated by Claude Code
  • Polls api.anthropic.com/api/oauth/usage every 5 minutes (community-discovered endpoint)
  • Color-coded status bar — gray (vs code background) / yellow / red based on usage percentage thresholds
  • Hover tooltip shows session + weekly quotas, reset countdowns, and last fetch time
  • No data collected — only a single request to Anthropic's own servers using your own token

Current State

  • Live on the VS Code Marketplace and in active personal use
  • Works automatically on VS Code startup with no configuration needed
  • Relies on an undocumented API endpoint — may break if Anthropic changes it

GearUp: Your Operational Readiness

Operational management platform for airsoft and milsim environments.

A mobile-first Progressive Web App designed for managing equipment, preparing loadouts, organizing operations, and tracking gear across devices.

Designed and directed by me, with implementation developed through AI-assisted workflow. System architecture decisions, UX design, feature planning, workflow logic, and product direction are my own.

Claude Cowork JavaScript HTML CSS IndexedDB Cloudflare Supabase
Read more about the project and development

Project Direction

    GearUp combines armory management and operational planning into one connected system built around real field workflow needs — offline-first, installable on phone and desktop
  • Built to solve real operational and equipment-management problems within airsoft and milsim environments
  • Evolved from a simple inventory tracker into a broader operations and preparation platform
  • Focused on practical workflow, low friction, and field usability instead of feature bloat
  • Armory and Operations Center systems are interconnected and designed to support each other
  • Built intentionally without frontend frameworks — lightweight structure, no build complexity, easy long-term maintenance

Technical Overview

  • Type: Progressive Web App (PWA) — installable, offline-capable, mobile-first
  • Frontend: Vanilla JavaScript, HTML, CSS
  • Storage: IndexedDB for local-first storage + Supabase PostgreSQL for cloud sync
  • Authentication: Cloudflare Access protected deployment with account mapping based on authenticated email
  • Operations Center: Briefings, objectives, ROE, notes, interactive maps, operational checklists, and field-focused planning tools
  • Loadout System: Automated loadout generation based on season, role, and equipment condition
  • Inventory System: Ownership tracking, lending/borrowing, maintenance intervals, ammunition tracking, and operational status management
  • Automation: Operation-linked loadouts, automatic checklist generation, maintenance reminders, operation history logging, and usage tracking
  • Import / Export: Excel and JSON support with duplicate detection and bulk import handling
  • Statistics: Chart.js visualizations for gear usage, categories, conditions, and operational history
  • Currency System: CZK / EUR support with live conversion across stored inventory values

Current State

  • Fully functional and in active personal use
  • Cross-device synchronization working through Supabase with automatic session handling
  • Offline-first architecture with local persistence and queued sync behavior
  • Private deployment hosted on Cloudflare Pages behind Cloudflare Access / Zero Trust protection
  • Currently expanding toward broader operational planning and long-term equipment lifecycle tracking
