Shot timer and training companion for airsoft and range training.

A mobile-first Progressive Web App built for use on the range, during the drill, in the moment — timer, drill library, and training log in a single offline-capable tool. Designed and directed by me, with implementation developed through AI-assisted workflow.

Claude Code JavaScript HTML CSS IndexedDB Cloudflare
Key Features
  • Offline-first PWA — installable on phone and desktop, no account required

  • Calibration system — measures ambient noise floor and shot peaks to suggest your threshold

  • Field Mode for outdoor readability in direct sunlight

Claude-meter: Limits at a glance

Claude Code usage limits, always visible.

A VS Code extension that displays your Claude Code usage limits (5-hour session and weekly session) as a permanent status bar item — always visible, no commands needed. Designed and directed by me, code written with AI assistance.

Claude Code JS JSON VS Code GitHub VS Marketplace
Key Features
  • Permanent VS Code status bar item — always visible, no commands needed
  • Tracks both 5-hour session and weekly Claude Code usage limits
  • Color-coded bar: neutral / yellow / red by usage percentage
  • Hover tooltip shows quotas, reset countdowns, and last fetch time
  • No data collected — single request to Anthropic servers using your own token
